Recent projects

Holidaze

Holidaze is my final, last year exam project at Noroff. It's a fictitious vacation venue booking site where the user can both register as a venue manager and list their own venues and/or book a listed venue themselves. This app was built using mainly React and Tailwind.

Best bargain

There are still improvements to be made and functionalities to be implemented.
